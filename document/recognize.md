### **购买新淘链**
### 请求URL:  
*/Mobile/Recognize/buy* 
### 请求方式:POST 
### 请求参数  
|参数|是否必填|说明|类型|
|--|--|--|--|
|numinput|是|购买数量|整数|
|recognize_id|是|众筹活动id|整数|
### 返回格式:JSON  
```
{
    "code": 0, //-1.余额不足 0.失败 1.成功  
    "msg": '提示信息',
    "data": null //成功时会有订单信息,注意保存订单id,支付时需要使用
    "jump": '' //code为-1时代表余额不足,需要跳转到充值url进行充值
}
```
